# Create YAML Nodes

Anemos provides various ways to create YAML nodes. It is possible to create all nodes using constructors.
It is also possible to create [Mapping](/docs/reference/yaml-manipulation/yaml-basics#mapping) and
[Sequence](/docs/reference/yaml-manipulation/yaml-basics#sequence) using the `parseMapping` and `parseSequence`
methods. These methods are useful when you want to create a node from a YAML string.
